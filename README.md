# client-portfolio-website

Premium portfolio starter built with Next.js, TypeScript, Tailwind CSS, and Framer Motion.

## Pages

- Home
- Projects
- About Me
- Contact

## Scripts

- `npm install`
- `npm run dev`
- `npm run build`
- `npm run typecheck`

## Structure

- `src/app`: app router pages and global styles
- `src/components`: reusable layout, navigation, motion, and project components
- `src/data/site.ts`: centralized portfolio content and navigation data
